
# ORBIS DEMO KIT

This demo kit includes all current materials for the Orbis DAM TFG project — a frequency-driven 3D visualizer for audio mastering.

## Structure:
- /code: Prototype source files
- /renders: Visual examples of frequency shape distortion (from Blender) and concept art
- /branding: Project logo and identity
- /documents: Current written report (Part 1 of the TFG)
